# MediCore-CPP

A console-based Hospital Management System developed in **C++** for the **Data Structures & Algorithms** course. The project demonstrates the practical application of object-oriented programming, file handling, and core data structures to manage hospital records through a menu-driven interface.

## Features

- Patient Management
- Doctor Management
- Appointment Handling
- Record Search & Updates
- File-Based Data Storage
- Menu-Driven Console Interface

## Technologies

- C++
- Object-Oriented Programming (OOP)
- Data Structures & Algorithms
- File Handling

## Getting Started

1. Clone the repository.
2. Open the project in your preferred C++ IDE.
3. Build and run the project.
